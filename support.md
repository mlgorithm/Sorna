# Sorna Support

If you need help with Sorna, please open an issue at [github.com/mlgorithm/Sorna/issues](https://github.com/mlgorithm/Sorna/issues).

## Please include

- device model
- iOS version
- app version
- steps to reproduce the issue
- screenshots, if relevant

## Common topics

### Papers are not loading

Check your network connection and try again. Some public sources may also be temporarily unavailable.

### Search results look incomplete

Search coverage depends on the availability and freshness of the upstream public data sources.

### PDFs or publisher pages open outside the app

Some paper links open in the browser or on a publisher site. Those destinations are controlled by the third party.

## Contact

For bug reports, support requests, and feature suggestions, use GitHub issues.
